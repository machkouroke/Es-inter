# Es-inter

## Configuration
- Executer le fichier de définition des données contenu dans le dossier `models/sql/shema.sql` pour créer la base de données.
- Executer le fichier `models/sql/user.sql` pour ajouter certains utilisateurs.
- Lancez un server php comme suit
```shell
php -S localhost:8000
```
- Accéder à [http://localhost/index.php?action=Users](http://localhost/index.php?p=Users)
